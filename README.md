## ğŸŒ¡ï¸ What is a Temperature Anomaly?
A temperature anomaly represents the deviation of the actual temperature from a reference long-term average. It is calculated as:

$$\text{Anomaly} = T_{\text{actual}} - T_{\text{baseline}}$$

Using anomalies provides a more accurate representation of climate change than absolute temperatures because it mitigates the impact of station location and elevation.
* **Positive Anomaly (+):** Warmer than the average.
* **Negative Anomaly (-):** Cooler than the average.
